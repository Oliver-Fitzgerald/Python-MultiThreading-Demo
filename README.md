# Python Multi-Threading Demo

The following project provides a walkthrough of multi-threading concepts via a marimo notebook
which details multi-threading concepts from the ground up with worked examples throughout.


# Depednencies

- python3
- uv


# Quick Start

Once the repo has been cloned run the following commands to run the notebook

`uv sync`

`source .venv/bin/activate`

`uv run marimo edit main_nb.py`
